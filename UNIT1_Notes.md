# AI AGENTS : 

- `IN SHORT` : An AI model capable of reasoning, planning, and interacting with its environment. </br> </br>
- `PRECISE DEFINITION` : An Agent is a system that leverages an AI model to interact with its environment in order to achieve a user-defined objective. It combines reasoning, planning, and the execution of actions (often via external tools) to fulfill tasks.</br>

- `AI AGENTS` Have Two Main Parts : </br>  </br>
   - 1.) `Brain (AI Model)` : This is where all the thinking happens. The AI model handles reasoning and planning. It decides which Actions to take based on the situation. </br> </br>
   - 2.) `Body (Capabilities and Tools)` : This part represents everything the Agent is equipped to do. The scope of possible actions depends on what the agent has been equipped with. For example, because humans lack wings, they can’t perform the “fly” Action, but they can execute Actions like “walk”, “run”, “jump”, “grab”, and so on. </br> </br>


- To summarize, an Agent is a system that uses an AI Model (typically an LLM) as its core reasoning engine, to:

     > `Understand natural language`: Interpret and respond to human instructions in a meaningful way. </br>
     > `Reason and plan`: Analyze information, make decisions, and devise strategies to solve problems. </br>
     > `Interact with its environment`: Gather information, take actions, and observe the results of those actions.
     

- Agent needs an AI Model at its core, and that LLMs are the most common type of AI models for this purpose.
